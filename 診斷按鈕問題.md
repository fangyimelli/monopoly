# 診斷按鈕問題

## 步驟 1：在瀏覽器控制台執行

```javascript
// 1. 檢查按鈕元素
const rollBtn = document.getElementById('rollDiceBtn');
console.log('按鈕:', rollBtn);
console.log('按鈕 disabled:', rollBtn?.disabled);
console.log('按鈕樣式:', window.getComputedStyle(rollBtn));

// 2. 檢查遊戲狀態
console.log('gameState:', window.game?.gameState);
console.log('isMyTurn:', window.game?.isMyTurn());

// 3. 手動啟用按鈕
rollBtn.disabled = false;
console.log('已手動啟用按鈕');
```

## 常見問題

### A. 按鈕被 CSS 隱藏
```javascript
const rollBtn = document.getElementById('rollDiceBtn');
console.log('display:', window.getComputedStyle(rollBtn).display);
console.log('visibility:', window.getComputedStyle(rollBtn).visibility);
```

### B. 按鈕被其他元素覆蓋
```javascript
const rollBtn = document.getElementById('rollDiceBtn');
const rect = rollBtn.getBoundingClientRect();
console.log('按鈕位置:', rect);
console.log('按鈕 z-index:', window.getComputedStyle(rollBtn).zIndex);
```

### C. 事件監聽器未綁定
```javascript
const rollBtn = document.getElementById('rollDiceBtn');
console.log('onclick:', rollBtn.onclick);
```
